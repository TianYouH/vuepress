# linux 命令

## 基础常用

1. 登录

```bash
ssh [用户名]@[IP]
```

2. 查看文件夹

```bash
ls    # 平铺查看
ll    # 列表查看
ls -a # 查看全部文件，包含隐藏文件
```

3. 新建文件夹

```bash
mkdir [文件夹名]
```

4. 删除文件夹

```bash
rm -rf [目录名称]
```

5. 修改 移动 文件夹

```bash
mv [文件] [目地]
```

6. 复制 文件夹

```bash
cp [文件] [目的]
```

7. 新建文件

```bash
touch [文件名]

vi [文件名] # vim 模式新建
vim [文件名] # vim 模式新建
```

8. 查看文件

```bash
cat [文件名]

head [文件名] # 打印前几行
tail [文件名] # 打印后几行
```

9. 查看文件内容

```bash
grep ["查找内容"] [文件名]
```

10. 学习vim命令

```bash
vimtutor
```
